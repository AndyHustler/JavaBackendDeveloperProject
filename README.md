Функционал пользователя
    1. Работа со своими задачами:
        просмотр списка своих задач: /task-api/myTasks
        создание новых задач; /task-api/create
        изменение своих задач; /task-api/update
        удаление своих задач; /task-api/delete
        изменение статуса задачи (например, «запланирована», «в работе», «сделана»); /task-api/update/{id}/status/{status}
        привязка задачи к группе задач и изменение группы. /task-api/update/{id}/grop/{grop}
    2. Работа с группами задач:
        просмотр списка своих групп задач; /task-group-api/curent-user-groups-list
        создание своих групп задач, /task-group-api/create
        переименование своих групп задач, /task-group-api/update
        удаление своих групп задач. /task-group-api/delete
    3. Регистрация с указанием почты и пароля.
    4. Авторизация с указанием почты и пароля уже зарегистрированного пользователя(в ответе возвращается JWT-токен).

Функционал администратора
    1. Просмотр списка пользователей. /user-api/users-list
    2. Просмотр всех задач в системе. /task-api/tasks-list
    3. Просмотр всех групп задач. /task-group-api/groups-list
    4. Просмотр задач выбранного пользователя. /task-api/user/{id}
    5. Просмотр простой агрегированной статистики(например, количество задач по статусам или по группам задач).
